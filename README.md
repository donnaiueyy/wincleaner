# WinCleaner

WinCleaner is a simple Python program designed to clear temporary files and free up system space automatically on Windows. It targets temporary files, the recycle bin, Windows update cache, and prefetch files to help keep your system running smoothly.

## Features

- Clears temporary files.
- Empties the recycle bin.
- Deletes Windows update cache.
- Removes prefetch files.

## Prerequisites

- Python 3.x
- Windows Operating System

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/wincleaner.git
   ```

2. Navigate to the project directory:
   ```bash
   cd wincleaner
   ```

## Usage

Run the `wincleaner.py` script using Python:

```bash
python wincleaner.py
```

This will start the cleaning process, and it will print the status of each cleaning step in the console.

## Warning

This program will permanently delete files from your system. Use it with caution and ensure that you do not require the files that will be deleted.
